微信小程序的一些坑

2017/7/17 posted in  tech

es6 转 es5, 如果遇到不小心重复 export 的情况

export {
    foo,
    bar,
    foo
}

转译过程不会报错, 但是最终生成的 js 似乎是变成了一个空文件, Network 请求正常, 没有 404, 但是控制台会报错 Uncaught Error: module "utils/API.js" is not defined

对于"腾讯内部的 KPI 项目"已经无力吐槽 🙂
